# ni4882
GPIB binding for Gauche.

Tested with GPIB-USB-HS on linux 2.6 with [Linux GPIB](https://linux-gpib.sourceforge.io/)
and macOS High Sierra with [NI488.2](http://www.ni.com/en-us/support/downloads/drivers/download.ni-488-2.html) 19.5
and macOS Monterey (12.1)  [NI488.2](http://www.ni.com/en-us/support/downloads/drivers/download.ni-488-2.html) 21.5.
