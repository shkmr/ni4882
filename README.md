# ni4882
GPIB binding for Gauche.
Tested on linux 2.6 with [Linux-gpib](https://linux-gpib.sourceforge.io/)
and macOS High Sierra with [NI4882](http://www.ni.com/en-us/support/downloads/drivers/download.ni-488-2.html) 19.5.

It is known that this does not work on macOS big sur, need to wait for NI to update NI4482 library.
